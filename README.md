# Audience-Sentiment-Analysis-I-Promised-You-The-Moon-IPYTM-
This project analyzes audience perception regarding character development in the Thai drama "I Promised You The Moon." Using *Web Scraping* and basic *Natural Language Processing (NLP)*, the script extracts hundreds of reviews from MyDramaList to determine if users believe the protagonists stayed true to their character or if there was an "out of character" shift in the sequel.

# Technical Stack:
* *Automation:* Selenium (Handling pagination & dynamic content).
* *Analysis:* Pandas (Data manipulation & Keyword Matching).
* *Visualization:* Matplotlib (Pie charts for sentiment distribution).
